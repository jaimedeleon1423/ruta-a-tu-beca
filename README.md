# Ruta a tu Beca

Plataforma educativa para estudiantes guatemaltecos que quieren obtener su beca de diversificado.

## Deploy en Vercel (5 minutos)

1. Sube este proyecto a GitHub (nuevo repositorio)
2. Ve a [vercel.com](https://vercel.com) → New Project
3. Importa el repositorio de GitHub
4. Click en Deploy (Vercel detecta Next.js automáticamente)
5. ¡Listo! Tu sitio está en línea

## Desarrollo local

```bash
npm install
npm run dev
```

Abre [http://localhost:3000](http://localhost:3000)

## Stack

- Next.js 15 (App Router)
- TypeScript
- Tailwind CSS
- Google Fonts (Plus Jakarta Sans + DM Sans)
- Lucide Icons

## Estructura

- `/` → Homepage con módulos y testimonios
- `/modulo/[1-4]` → Los 4 módulos del recorrido
- `/guia/[1-12]` → Las 12 guías con contenido completo
- `/becas/vista-hermosa` → Página destacada de la beca
- `/becas` → Directorio de becas
- `/guias` → Índice de todas las guías
- `/acerca` → Información del proyecto
